Right Here, Right Now

Interactive Geographic Data Visualization Tool

Project description

The application helps users make decisions about their travel plans throughout Chicago: after a user has selected his trip preferences, the app shows on the interactive map a set of suggested layers. At any time the user can also add/remove layers according to the real-time data he wants to see. All the data sources are shown with respect to a restricted area of the city, based on 2 different modes: rectangular selection and custom path.
